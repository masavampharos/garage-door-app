# Garage Door App for Kids

## Overview
This project is a simple and interactive web application designed to help children learn about garage door mechanics in a fun way. The app simulates the opening and closing of a garage door, complete with sound effects and a day/night mode toggle.

## Features
- Simple and intuitive interface
- Animated garage door opening and closing
- Sound effects (with toggle option)
- Day/Night mode switch
- Responsive design for various devices including smartphones

## File Structure
- `index.html`: The HTML structure of the application
- `styles.css`: CSS file defining the application's styles
- `script.js`: JavaScript file controlling the application's behavior
- `garage_open.mp3`: Audio file for the door opening/closing sound
